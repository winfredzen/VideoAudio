# 图像基础

视频：

1.视频由图像组成

2.为了传输或者占用更小的空间而被压缩

3.最终在显示设备上显示（未被压缩）

图像：

1.由像素组成

2.像素由RGB组成

3.分辨率 - x,y由多少个像素点

4.像素的位深

+ RGB888（24位）
+ RGBA（32位）
